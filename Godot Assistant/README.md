# Godot AI Assistant

Free AI coding sidekick for Godot 4.

## Features
- **One-Click Insert:** AI generates code, you click 'Insert' to put it in your script.
- **FREE AI:** Uses OpenRouter (Llama 3.2, Gemma 2, Mistral).
- **Auto-Router:** Scans 13+ free servers if one is busy.
- **Plug & Play:** API key built-in.

## Install
1. Copy the `addons` folder to your project root.
2. Go to **Project Settings -> Plugins** and check **Enabled**.
3. Click the **AI Assistant** tab at the top!
